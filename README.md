## Dependencies

- Azure Key Vault
- Azure Storage Account
  - Azure Storage Queue

## Development

### Installing dependencies

Install the dependencies with:

```shell
npm install -D
```

### Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Installing pre-commit hooks

Some basic code quality checks are run before every commit. To install the pre-commit hooks, run:

```shell
npm run prepare
```

### Testing

Run the tests with:

```shell
npm run test
```
